Project Nmae : AI Review Sentiment Web App | Python, Streamlit, NLP
Work:
Developed and deployed a real-time sentiment analysis web application to classify user reviews as positive or negative.
Implemented TF-IDF vectorization and Logistic Regression for accurate text classification.
Integrated confidence scoring to improve prediction interpretability and user trust.
Designed an interactive UI using Streamlit and deployed the application for real-time usage.
